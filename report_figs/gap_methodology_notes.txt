GENERALIZATION GAP -- METHODOLOGY NOTES
=======================================
Generated by merge_all_gaps.py & compare_all_optimizations.py
Feature set: Freq-72 (TD + spectral; 72 features per window)
Normalization: per-subject z-score (best across all models)
Updated: March 2026

EVALUATION PROTOCOL
-------------------
All three model families (RF, SVM, CNN) follow the same two-paradigm evaluation:

  SD  (subject-dependent):  Per-subject 5-fold cross-validation.
      Each subject's windowed data (~650-850 windows) is split into 5 folds.
      A separate model is trained for each subject using their own data only.
      Result: one metric per subject, averaged across n=40 subjects.

  LOSO (leave-one-subject-out):  Cross-subject generalisation.
      Each subject is held out in turn; the model trains on all 39 remaining subjects.
      Per-subject z-score normalization applied within the LOSO loop (leak-free).
      Result: one metric per held-out subject, averaged across n=40 subjects.

The generalization gap (Delta F1 = F1_SD - F1_LOSO) quantifies the performance
cost of not having labelled data from the target subject.

All three models show a substantial generalisation gap.
The LOSO performance with per-subject normalization converges to a narrow range
(RF=0.773, SVM=0.777, CNN=0.754),
indicating that cross-subject variability is the primary bottleneck
regardless of model architecture.

OPTIMISATION JOURNEY
--------------------
  Stage 0 (Baseline, global norm):  SVM=0.708, RF=0.722, CNN=0.682
  Stage 1 (Per-subject norm):       SVM=0.777, RF=0.773, CNN=0.754
  Stage 2 (Feat sel / augment):     SVM=0.780, RF=0.773, CNN=0.760
  Stage 3 (Ensemble):               SVM+RF+CNN = 0.792

Per-subject normalisation is the dominant intervention, contributing 5-7 pp gains
across all models. Feature selection provides marginal improvement for SVM only;
CNN augmentation provides marginal improvement. Ensemble voting delivers the
final +1.2 pp gain by leveraging complementary error patterns.

NOTE ON NORMALIZATION
---------------------
Per-subject z-score normalization applied channel-wise within the LOSO loop.
This directly addresses inter-subject amplitude variability.
Improvement over global baseline: SVM +0.069, RF +0.051, CNN +0.072 F1.
All improvements are statistically significant (Wilcoxon p < 0.0001).
Effect sizes: SVM d=1.48, RF d=1.15, CNN d=1.34 (all large).

NOTE ON FEATURE SELECTION
-------------------------
Evaluated: RFE and MI at 36 and 27 features, under both per-subject and global norm.
SVM: RFE-36 yields marginal +0.003 (p=0.38, not significant).
RF: Feature selection consistently degrades performance (RFE-36 p<0.001, -1.3 pp).
CNN: Not applicable (operates on raw signals, not hand-crafted features).

NOTE ON CNN AUGMENTATION
-------------------------
Four augmentation strategies tested under per-subject normalisation:
  Gaussian noise: +0.6 pp (p>0.05, not individually significant)
  Channel dropout: -0.3 pp
  Time masking:    -1.0 pp
  Combined:        -0.9 pp
The CNN has sufficient implicit regularisation from dropout and batch normalisation;
augmentation provides diminishing returns.

NOTE ON ENSEMBLE
----------------
Hard majority vote evaluated for 4 combinations:
  SVM+RF:      0.777 (matches SVM individual)
  CNN+SVM:     0.777 (matches SVM individual)
  CNN+RF:      0.773 (matches RF individual)
  SVM+RF+CNN:  0.792 (best overall, +1.2 pp over best individual)

The 3-way ensemble exploits complementary error patterns:
RF and SVM agree on most easy subjects but diverge on hard ones;
CNN makes different mistakes than classical models on movement boundaries.

NOTE ON HYPERPARAMETERS
-----------------------
Classical SD: Fixed optimal hyperparameters from sweep (SVM C=10 scaled, RF n_estimators=500).
Classical LOSO: Nested inner CV (5-fold GroupKFold) per LOSO fold to select optimal
hyperparameters. This is conservative -- the LOSO model is better-tuned per fold,
meaning the reported gap is a lower bound on the true generalisation cost.
CNN: Identical SimpleEMGCNN architecture (3 conv blocks: 32->64->128,
kernel_size=9/7/5, AdaptiveAvgPool1d, Dropout=0.25) for both SD and LOSO.

NOTE ON STATISTICAL TESTING
----------------------------
All pairwise comparisons use Wilcoxon signed-rank tests on per-subject F1 scores
(N=40 paired observations). Effect sizes reported as Cohen's d.

End-to-end (Stage 0 → Stage 3 ensemble):
  SVM: p < 0.0001, d = 1.58 (very large)
  RF:  p = 0.00001, d = 0.75 (medium-large)
  CNN: p < 0.0001, d = 0.97 (large)
